# streamlit_rtsp
show rtsp stream in streamlit web via ffmpeg, You can set the hardware acceleration manually set it in the code.

1. Dependent environment:
sudo apt install ffmpeg
pip3 install streamlit opencv-python

2. run test:
python3 test.py
